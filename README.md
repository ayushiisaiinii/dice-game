dice-game
🎲 Dice Game Web App

A simple and interactive dice game built using HTML, CSS, and JavaScript. On each page refresh, two virtual dice are rolled randomly for Player 1 and Player 2. The app dynamically updates the dice images and displays the winner based on the generated values.

🚀 Key Features:

Random dice roll generation using JavaScript
Dynamic image updates based on outcomes
Real-time winner display logic
Clean and responsive UI design
🛠️ Tech Stack:

HTML5
CSS3
JavaScript (DOM Manipulation)
🎯 How it works: Refresh the page to roll the dice. The player with the higher number wins. If both values are equal, it results in a draw.

💡 This project demonstrates core JavaScript concepts like random number generation, DOM selection, and event-driven UI updates.
